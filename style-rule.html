Style Rule

These are rules that you have to remember while writing styles for your web pages(html documents)

        /* Here we call "p" as selector */
        /* Here we call "color" as property */
        /* Here we call "red" as value to "color" property */
        /* Here we call "color:red;" as a declaration */
        /* Here we call "p { ... }" as a rule*/
        p {
            color: red;
            background-color: white;
        }
        
        /* Here we call "px" as a unit */
        p {
            font-size: 20px;
        }
